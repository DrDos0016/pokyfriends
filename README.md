# Pokyfriends

This is the main django project for my personal website. Projects for this
site are given their own repositories.

Although I aim to make these other projects function as Django apps that can
be used with any Django site, they are developed as part of Pokyfriends and
likely need at least some modifications to integrate onto another site.

Pokyfriends itself isn't exactly meant to be used by others and you'll likely
have to adjust various files and settings in order to run your own copy.

## List of Applications

* pokyfriends_web - "Pokyfriends Web", The main site and the only one included in this repo.
* bk_mario_maker - "Blitzkrieg Mario Maker", a tool for Super Mario Maker to add
limitations and encourage groups to make their own levels with the same limits.
* blackout - "Blackout Poetry", a very old JS bookmarklet to let you make
Blackout poetry by blacking out text on other pages.
* blog - "Blog", an extremely simplistic blog platform.
* chimp_or_chump - "Chimp Or Chump", an old HTML5 game to challenge your short
term memory
* clock - "Gen5 Pokemon Clock", a simple JS clock that shows a different Pokemon
for every minut of the day.
* gallery - "Gallery", a tool for uploading, collecting, and exhibiting
artwork.
* gen4_wifi - "Gen4 Wifi", a recreation of the DS era Pokemon games' wifi
connection screen
* maestro_tiles - "Maestro Tiles", a proof-of-concept for a music-themed puzzle
game
* owo_whos_this - "OwO, Who's This?", a PICO-8 game created for a game jam
* pmdexplorers - "Pokemon Mystery Dungeon Explorers Of", a tool to generate
fake boxart for the Nintendo DS Pokemon Mystery Dungeon games
* pokedex_ebooks_web - "Pokedex Ebooks - Web", a markov chain generator that
used Pokedex entries from the Pokemon games as source data.
* pokemon_size_quiz - "Pokemon Size Quiz", a quiz game to guess the heights of
Pokemon according to the Pokedex.
* pokemon_type_chart_quiz - "Pokemon Type Chart Quiz", a quiz game to fill out
the Pokemon type-matchup chart.
* pokepan_crystal - "Pokepan Crystal", a romhack of Pokemon Crystal to remove
the check for gender when using the move Attract.
* stickers - "Stickers", a page to display various Telegram stickers.
* stitchr - "Stitchr", a tool to grab images from a Tumblr post (or local files)
and stitch them together into a single large composite image.
* train_box_release - "Train-Box-Release", a quiz that selects three random
Pokemon and asks which one you'd train, which one you'd put in a box, and which
one you'd release
* zzt - "ZZT", a simple way to host browser-playable ZZT worlds that uses the
Zeta emulator
